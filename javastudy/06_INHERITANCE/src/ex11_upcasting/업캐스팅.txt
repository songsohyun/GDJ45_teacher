업캐스팅

1. upcasting
2. 자식 클래스 타입의 인스턴스가 부모 클래스 타입으로 변환되는 것을 말한다.
3. 자식 클래스 타입의 인스턴스를 생성할 때 부모 클래스 타입에 저장할 수 있다.
4. 자동으로 변환되서 저장된다.
5. 부모 클래스 타입으로 저장하는 경우 부모 클래스가 가지고 있는 메소드만 사용할 수 있다.
6. 형식
    class Parent {
        void a() { }
    }
    class Child extends Parent {
        void b() { }
    }
    
    Parent instance = new Child();
    instance.a();  // 가능
    instance.b();  // 현재 불가
