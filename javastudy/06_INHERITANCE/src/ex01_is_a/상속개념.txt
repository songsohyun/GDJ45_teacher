상속

1. inheritance
2. 어떤 클래스의 기능을 다른 클래스에게 상속해 주는 것이다.
3. 부모 클래스의 기능을 자식 클래스에게 상속해 주는 것이다.
4. 부모 클래스
   : 슈퍼 클래스(super)
5. 자식 클래스
   : 서브 클래스(sub)
6. 상속이 가능한 관계
    1) IS-A  관계 : 자식 IS A 부모.  연필은 학용품이다. TV는 전자제품이다.
    2) HAS-A 관계 : 자식 HAS A 부모. 철수는 연필을 가지고 있다. 군인은 총을 가지고 있다.
7. 형식
    class 부모클래스 {
    
    }
    class 자식클래스 extends 부모클래스 {
    
    }
